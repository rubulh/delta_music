Enable Apache Rewrite Module on Linux

April 28, 2011

pakcom Computer Apache, Linux 3 Comments

I use Linux Mint 10 “Julia”. After I installed apache, rewrite module was disabled by default. We need to do some configuration to enable it. It’s easy

Here I describe the steps to enable rewrite module (mod_rewrite)

Move to apache folder. Type :

view source
print?
1cd /etc/apache2

Enable mod_rewrite. Type :

view source
print?
1sudo cp mods-available/rewrite.load mods-enabled/

Then edit overriding configuration on /etc/apache2/sites-available/000-default.

We can edit it with editor like nano or gedit, suit your self . If you choose nano editor, you could do it by type :

view source
print?
1sudo nano /etc/apache2/sites-available/000-default

Find the following :

view source
print?
1Options Indexes FollowSymLinks MultiViews
2AllowOverride None
3Order allow,deny
4allow from all

and change it to :

view source
print?
1Options Indexes FollowSymLinks MultiViews
2AllowOverride all
3Order allow,deny
4allow from all

Finally, restart  your Apache by type

view source
print?
1sudo /etc/init.d/apache2 restart

Done!
